# ubuntu-setup
Run the bash file to set-up freshly installed Ubuntu desktop  
Packages and Softwares installed in this sh file  
1. Slack  
2. Git (includess configuration)
3. Chrome
4. Anydesk  
Packages installed by default:  
1. nvtop
2. tmux
3. net-tools
4. openssh-server
5. VSCode
6. Gnome Tweaks

To start set-up process
```sh setup.sh```
